# lecture_kinetic_brownian

ブラウン運動

講義「分子運動論」で利用しうる

その他、講義「高分子物理学特論」（R5）でも活用
その際、理想鎖やFlory-Huggins鎖のブラウン運動にも拡張
さらに、レプテーションの簡単なモデルとしても拡張

SAW鎖への拡張はそちらで継続するとして、こちらではブラウン運動の本質に迫るような拡張を行なっていきたい。
3dへの拡張も行いたい。

＜260201コメント＞
1dのアニメーションが1dのそれというより、x-tグラフのようになっていたのでv1として修正した  
1dと2dを比較すると、どちらもMSD = <d^2> = tとなっており次元性が現れない  
ここではD = 1/2 (1d), 1/4 (2d)となるとして、良しとしているが、チェックが必要  

＜260401コメント＞  
3dも作ったけど同じ。ここではD = 1/6 (3d)となると考える。  

格子ランダムウォーク（ブラウン運動）では、N個のステップが、2dではx,yの2方向に等分される  
3dではx,y,zの3方向に等分される  
そのため結果的にはD = 1/2dとなり、MSD = tが得られることになる  
つまり振る舞いは正しい
